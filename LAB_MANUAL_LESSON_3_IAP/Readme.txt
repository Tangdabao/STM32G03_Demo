//*****************************************************************************************//
RESET PIN ：
- 模式选择：
RESET有三种模式：1.Reset功能，内部Reset不会导致外部IO拉低
				 2.普通IO功能（PF2），可以作为IO、MCO
				 3.双向复位：被配置为复位模式

- 中断模式需要注意
发现如果设置为下降沿中断，上电会在main之前先进中断

//*****************************************************************************************//
使用芯片自带的Bootloader
				BOOT_LOCK bit		nBOOT1 bit	BOOT0 pin	nBOOT_SEL bit		nBOOT0 bit
System memory		0					1			1			0					X	（硬件IO）
	||
System memory		0					1			X			1					0  （软件办法）

推荐软件方法，只需修改 nBoot0 bit为0
